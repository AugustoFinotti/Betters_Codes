HTML

<html> </html>

HTML - Tag root, em português: raiz, que da inicio ao conteúdo de fato, outras tags são inseridas dentro dela.

<head> </head>

Head - Defini informações e configurações da página, atributos que não são visívieis ao usuário do site como: título, configuração de caracteres, links etc.

<body> </body>

Body - Parte do site em que se concetra a parte do conteúdo de fato, geralmente se destina ao que é visível dentro da página html.

<h1> </h1>

Heading 1 - 1° cabeçalho.

<h2> </h2>

Heading 2 - 2º cabeçalho.

<h3> </h3>

Heading 3 - 3º cabeçalho.

<h4> </h4>

Heading 4 - 4º cabeçalho.

<h5> </h5>

Heading 5 - 5º cabeçalho.

<h6> </h6>

Heading 6 - 6º cabeçalho.

<p> </p>

Paragraph - usada para inserir textos nos sites.

<div> </div>

Div - Delimita uma divisão dentra sua página html, delimita uma região onde posteriormente poderão ser aplicadas configurações desejadas.

<article> </article>

Article - é mais uma das tags semânticas, essa é usada para indiar aos motores de busca que o conteúdo envolvido pela tag se trata de um artigo.

<!--   -->

Comentário - dentro dessa tag você pode colocar o que desejar, pois não influenciará no código, ou seja, será ignorado pelo intretador do código html. Geralmente são usados para comentar algo sobre o prórpio código, os comentários, não só html, fazem parte de uma boa conduta como desenvolvedor, além de tornar o código mais compreensível e editável posteriormente em possíveis manutenções.

<var> </var>

Variable - "estiliza" o texto como se fosse uma variável, possúi valor semântico também. Pode ser usada junto com as tags <code> e <kbd>
